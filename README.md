# Paradise Nursery

## Overview
Paradise Nursery is a dynamic e-commerce web application built for plant enthusiasts. The platform allows users to browse an extensive catalog of indoor, outdoor, and succulent plants, view plant details, and manage their shopping cart effectively. Built with modern web development practices, Paradise Nursery offers a clean, responsive, and intuitive shopping experience.

## Features
- **Interactive Landing Page**: A visually appealing welcoming page emphasizing the brand.
- **Product Catalog**: Easily browse plants categorized by type (Indoor, Outdoor, Succulents).
- **Shopping Cart**: Add items, increase/decrease quantities, remove items, and view total cart costs in real-time.
- **State Management**: Robust state handling utilizing Redux Toolkit.
- **Responsive Design**: Beautiful interface tailored for desktop, tablet, and mobile devices.

## Tech Stack
- **Frontend Framework**: React (Functional Components with Hooks)
- **State Management**: Redux Toolkit & React-Redux
- **Routing**: React Router DOM (v6)
- **Styling**: Custom CSS with CSS Variables and Flexbox/Grid
- **Icons**: Lucide React
- **Build Tool**: Vite

## Setup Instructions

To run this project locally, please follow these steps:

1. **Clone the repository** (if applicable) or download the source code.
2. Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
3. Open your terminal and navigate to the project directory:
   ```bash
   cd paradise-nursery
   ```
4. Install the required dependencies:
   ```bash
   npm install
   ```
5. Start the development server:
   ```bash
   npm run dev
   ```
6. Open your browser and navigate to the local URL provided by Vite (typically `http://localhost:5173/`).

Enjoy exploring Paradise Nursery!
